Main screen
====================
![MainScreen]{:width="40%"}

Below the map area there are five buttons

When a site is shown on the map, selecting the site icon allows for:
- Displaying further site information (including site access documentation)
- Navigation to the site

![Bilden] Sites
---------------
Shows all sites (can take a while if you have many sites in a zoomed out state).

Areas
-----
Shows all location areas/tracking areas
When an area is shown. Sites in that area can be shown by selecting the area icon

Trash
-----
Emptys the screen

Settings
-------
Takes you to the settings page

Search
------
Search for site or address (long press)

Siri can also be uset to search for sites using spoken language
<b>Hey Siri - "Ranviewer search for site"</b>
